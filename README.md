**基于机器学习的可视化学习平台**

实现机器学习基本算法

其中包括：KNN、ID3、C4.5、SVM、朴素贝叶斯、Kmeans

除此之外，还有基本的数据处理如标准化、归一化、分箱等操作

采用Django框架进行网页端节点拖拽实现各个算法

前端使用JSplumb.js实现拖动，使用Ajax.js实现数据传输
